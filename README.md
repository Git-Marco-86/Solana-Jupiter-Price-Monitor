# Solana Jupiter Price Monitor

A TUI (Text-based User Interface) written in Rust to monitor token prices and setting alerts on the Jupiter (Solana) platform.

---

### The Story

I built this little script for my own personal use a few months ago to keep an eye on some Solana tokens. This was a fun side-project to practice my Rust skills.

### Usage

1. Clone the repository.
2. `cd Solana-Jupiter-Price-Monitor`
3. `cargo run`
4. Follow the on-screen prompts to enter a token address and set your alerts.
